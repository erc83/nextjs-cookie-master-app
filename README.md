# Next.js Cookie Master


* contruir los modulos de node y levantar Next

```
yarn install
yarn dev
```


* Para levantar en forma de produccion
```
yarn build
yarn start
```



## Ir a la url base una vez levantada la app

LLamar:
```
    http://localhost:3000
```
